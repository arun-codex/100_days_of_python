## Key Concepts

### Variables as Containers
A variable in Python is like a container that stores data.
- **Red Container** = Sugar
- **Blue Container** = Salt

This is what variables mean in simple terms. They hold data of different types.

For example, when we write:
```python
a = 1
b = 2
print(a + b)
```
Here, `a` and `b` are variables storing the integer values `1` and `2`, and we print their sum.
